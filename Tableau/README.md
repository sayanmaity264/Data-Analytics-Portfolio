# Airbnb Dashboard (Tableau)

## Project Overview
This dashboard analyzes Airbnb listings and provides insights into pricing, bedroom distribution, revenue trends, and location-based pricing.

## Tools Used
- Tableau
- CSV Dataset

## Dashboard Features
- Average Price by Bedroom
- Property Count by Bedroom
- Price by Zipcode
- Revenue per Week
- Interactive Filters
